## LiveFrame v1.0.1

### RU
Первый публичный релиз LiveFrame.

**Что внутри:**
- Музыкальный оверлей для OBS (YouTube playlist + single video)
- Twitch chat overlay с автофильтрацией и role-фильтрами
- Games overlay (CS2 Faceit stats по nickname/profile link, без API key)
- Пресеты стилей, live preview, auto-refresh
- Stream controls: Go Live / Pause / End / Blank
- First-run wizard + pre-stream checklist
- Portable Windows build (без установки)

### EN
First public release of LiveFrame.

**Included:**
- Music overlay for OBS (YouTube playlist + single video)
- Twitch chat overlay with safe filtering and role filters
- Games overlay (CS2 Faceit stats by nickname/profile link, no API key)
- Style presets, live preview, auto-refresh
- Stream controls: Go Live / Pause / End / Blank
- First-run wizard + pre-stream checklist
- Portable Windows build (no installer)

### Download
- `LiveFrame-win32-x64.zip` (extract and run `LiveFrame.exe`)
